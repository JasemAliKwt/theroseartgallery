Bank Accounts Managment
.

Using C++, I built a program that stores and manages accounts for a bank. The program allows the client to create an account, and then give them the option to create checking and/or saving accounts. I had to debug and test my code to make sure that all cases were taken into account.

I learned alot from this project, and I learned how to make it more effecient by using dictionaries.
Throughout the project, I honed my skills such as:

C++
Used C++ to implement the entire program.
Testing
Tested multiple cases in order to verify that all cases were taken into consideration and that the program will not crash no matter the input.
Global/Private Variables
Learned the differeces between local, global, and private variables and how to use them.
