Building a MIPS to Binary Converter .

Using Pything in the Linux operating system, I first started planning out my converter by creating a flow chart and dictionaries. I learned the differences between MIPS and ARM assembly languages. I also learned how to manually convert assembly instruction to machine language. Using my knowledge in those topics, I organized my dictionaries and loops in a way where the user would input a MIPS file, and the program would create a file with the converted results. I had to test many different cases to make sure that the program would not run into crashes or accidentally get stuck in an endless loop.
